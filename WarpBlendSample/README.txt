The included files can be used for the Warping and Intensity adjustment API.

What's included:

NVIDIA NVAPI Interfaces for per pixel Intensity and Warping.pdf
 - brief document explaining the new API

SampleWarpBlend
 - Sample implementation code

Requirements:
 - NVIDIA Quadro 1200 class or higher products with Fermi, Kepler, Maxwell or newer GPUs
 - NVS 810 & 510

 - Windows Quadro Display drivers 302.82 or newer
